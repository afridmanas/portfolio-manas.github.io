Network Intruder Detective System

   

The Network Intruder Detective System is a project developed using Cisco Packet Tracer. It is designed to detect and prevent unauthorized access to a network by monitoring network traffic and identifying potential intruders. The system uses a combination of network security protocols and machine learning algorithms to analyze network traffic and identify suspicious activity. When an intrusion is detected, the system sends an alert to the network administrator, who can take appropriate action to prevent further damage. The Network Intruder Detective System is an effective tool for protecting networks from cyber attacks and ensuring the security of sensitive data.

SLATE Management System Mobile App

The SLIATE management system developed using Flutter is a comprehensive software solution designed to streamline and automate various administrative tasks in SLIATE. The system includes features such as course management, grade management, and faculty management. It also includes a Newsfeed Page to fetch on from Website, faculty, and administrators. The system is user-friendly and can be accessed from any device with an internet connection. With this system, college administrators can easily manage and monitor various aspects of the college, while students can access their academic information and communicate with their peers and faculty members.

BMI Calculator Mobile App

The BMI Calculator project developed using Flutter is a simple yet useful application that allows users to calculate their Body Mass Index (BMI) based on their height and weight. The application has a user-friendly interface that makes it easy for users to input their height and weight and get an instant calculation of their BMI. The BMI Calculator project is a great example of how Flutter can be used to develop simple and useful applications that can be used by anyone.
